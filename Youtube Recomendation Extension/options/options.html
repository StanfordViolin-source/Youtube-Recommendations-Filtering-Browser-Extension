YouTube Music-Only Filter

This page controls the keyword lists used to decide if a recommendation is music or not. Each line is one keyword or phrase.

Strong Music Keywords

If any of these appear in the title or channel name, the item is treated as music.

Moderate Music Keywords

These are weaker signals. They count as music unless the duration is extremely short or long.

Non-Music Keywords

If any of these appear in the title or channel name, the item is blocked immediately.

Channel Music Tokens

If the channel name contains one of these AND the duration is in the typical music range, it is treated as music.

Default Policy When Unsure

Lower values scan faster; higher values reduce CPU usage.
